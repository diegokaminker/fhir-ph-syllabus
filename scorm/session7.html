This session covers FHIR Implementation Guides (IGs), which provide specific guidance for implementing FHIR in particular use cases. We'll explore how IGs constrain and extend FHIR resources for specific domains.

Learning Objectives

By the end of this session, you will be able to:

  • Understand the purpose of Implementation Guides
  • Navigate and use FHIR IGs
  • Apply IG constraints to resources
  • Identify public health specific IGs
  • Implement IG-based solutions
